# Taski

Веб-приложение для управления задачами с возможностью создания, редактирования, удаления и фильтрации по статусу выполнения.

## Технологический стек

### Backend
- **Django** 3.2.3 — веб-фреймворк
- **Django REST Framework** 3.12.4 — REST API
- **django-cors-headers** 3.13.0 — обработка CORS
- **SQLite** — база данных
- **Token Authentication** — аутентификация

### Frontend
- **React** 18.2.0 — UI библиотека
- **Axios** 1.2.1 — HTTP клиент
- **Bootstrap** 5.2.3 — стилизация
- **Reactstrap** 9.1.5 — React компоненты Bootstrap

## Архитектура

Проект построен по архитектуре клиент-сервер с разделением на backend (Django REST API) и frontend (React SPA). Backend предоставляет RESTful API для управления задачами, frontend реализует пользовательский интерфейс с модальными окнами и фильтрацией задач.
